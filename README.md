# wiShell (Windows Shell)
A Windows shell (explorer.exe) replacement

## Feautres
* Modular
* Easy to use

## Modules
### wiCore
- Core system functionality
- ~10KB disk usage

### wiPanel
- Editable panels/taskbars
- ~?KB disk usage

### wiWM
- Customizable window manager
- ~?KB disk usage

### wiTerm
- Custom terminal/command prompt
- ~?KB disk usage

### wiConf
- Configuration manager for wiShell
- ~?KB disk usage
